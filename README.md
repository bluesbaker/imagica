# imagica
Image to ASCII

## Build
```bash
$ go build -o .\bin\imagica .\imagica.go
```

## How to use?
Example:
```bash
$ ...\imagica image.jpg -cols 200 -rows 100
```

Flags:
```
-source string
    image source
-cols int
    columns (default 80)
-rows int
    rows (default 40)
```
